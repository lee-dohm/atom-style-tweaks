    list_tweaks(options \\ [])

    View Source

    Lists the tweaks according to the options.

    Defaults to listing all original (not forked) tweaks, newest first.

    Options

    • :for - User whose tweaks to list
    • :forks - When true, includes forked tweaks in the list (default: false)
    • :type - Includes only tweaks of the given type, if nil lists all tweaks (default: nil)
