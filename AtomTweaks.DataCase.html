    Atom Tweaks v0.1.0+c1f5afa AtomTweaks.DataCase View Source

    This module defines the setup for tests requiring access to the application's data layer.

    You may define functions here to be used as helpers in your tests.

    Finally, if the test case interacts with the database, it cannot be async. For this reason, every test runs inside a transaction which is reset at the beginning of the test unless the test case is marked as async.
